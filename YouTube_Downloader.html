yt-dlp — Usage Guide (Collapsible Commands) & Video Player Example

What is yt-dlp?

yt-dlp is a feature-rich, open-source command-line tool for downloading videos and audio from YouTube and over one thousand other websites. It is a maintained fork of youtube-dl with active development and frequent compatibility fixes.

Official repository: https://github.com/yt-dlp/yt-dlp

Installation (Quick Reference)

  • Standalone binary (recommended): Download from GitHub Releases
  • Python (pip): python3 -m pip install -U "yt-dlp[default]"
  • Update: yt-dlp -U (or yt-dlp --update-to nightly for latest fixes)

Command Examples

General syntax: yt-dlp [OPTIONS] "URL"

**Summary:** Basic downloads (video + audio)

Default behavior — best quality video + best audio, automatically merged:

yt-dlp "https://youtu.be/VIDEO_ID"
**Summary:** Video-only or audio-only downloads
GoalCommand
Best video only (no audio)yt-dlp -f bestvideo "URL"
Best audio only (usually m4a / opus)yt-dlp -f bestaudio "URL"
Extract audio → convert to MP3yt-dlp -x --audio-format mp3 "URL"
**Summary:** Inspect available formats first

List all video/audio formats with resolution, bitrate, codec, etc.:

yt-dlp -F "URL"

Then download a specific combination (example: 1080p video + best audio):

yt-dlp -f 137+140 "URL"
**Summary:** Playlists & channels

Download entire playlist or channel:

yt-dlp "PLAYLIST_OR_CHANNEL_URL"

Download only videos from index 5 to 15 (inclusive):

yt-dlp --playlist-start 5 --playlist-end 15 "URL"
**Summary:** Subtitles & metadata

Download auto-generated English subtitles (if available):

yt-dlp --write-auto-sub --sub-lang en --convert-subs srt "URL"
**Summary:** Custom output filename

Common flexible template:

yt-dlp -o "%(upload_date)s - %(title)s [%(id)s].%(ext)s" "URL"
Quick reminders
• Enclose URLs in quotes when they contain special characters.
• Run yt-dlp --help or visit the GitHub README for advanced options.
• Update frequently — many sites change their internal structure.
